Import new capabilities for the delivery department

New ability import

New capability import description

When delivery department need the new techniques that has not yet been applied to HSBC project, capability center can import this new ability in several ways:

1.Contact external resources: after the corresponding assessment, capability center can contact outside training resource to cultivate this new ability for specific personnel.

2.Study from internal staff: if the assessment found out it is not suitable to invite external resources, capability center will organize personnel to study first, and then teach the project teams.

Currently capability center has successfully imported many new abilities, including mulesoft, angularJs, etc.
