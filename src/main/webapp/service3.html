TDD

    TDD is an acronym for a core practical technology and design methodology in agile development named Test-Driven Development. To apply TDD, before developing function code, user should write unit test case code and test it to decide which product code should be written.

    You can obtain professional TDD service from this channel: Agile Service Catalogue --Devops Full-stack Tool--Tool Training-- TDD.

BDD

BDD

BDD

    BDD is the acronym of Behaviour-Driven-Development, the evolution of BDD. As a design method, it can improve design effectively and guide team through the system evolution process.

    BDD base on a kind of all-purpose language. This language is used both by clients and developers to define system behaviour. Clients and developers using same language to describe same system can avoid the problem resulted from different expression at the extreme.

You can obtain professional BDD service from this channel: Agile Service Catalogue --Devops Full-stack Tool--Tool Training-- BDD.

Integration Environment Building
